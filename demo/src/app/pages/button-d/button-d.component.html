Share Button Directive

Overview

[shareButton] is a directive that converts its element to a share button.

Installation

Install the following packages

NPM

$ npm install --save @ngx-share/core @fortawesome/fontawesome-free-webfonts

YARN

$ yarn add @ngx-share/core @fortawesome/fontawesome-free-webfonts
Import the styles

Import FontAwesome for the icons "app/src/style.scss" 


@import "~@fortawesome/fontawesome-free-webfonts/css/fa-brands.css";
@import "~@fortawesome/fontawesome-free-webfonts/css/fa-solid.css";
@import "~@fortawesome/fontawesome-free-webfonts/css/fontawesome.css";
Import the module

The most basic share button needs only the name of the social network. e.g: shareButton="twitter"

You can get the official colors and icons by injecting ShareButtons service in your component
